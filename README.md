## 🚦 Getting Started

### 1️⃣ Clone Repo
```bash
git clone https://github.com/your-username/starks-hub.git
cd starks-hub
````

### 2️⃣ Install Dependencies

```bash
# Frontend
cd frontend
npm install

# Backend
cd ../backend
npm install
```

### 3️⃣ Run Locally

```bash
# Backend
npm start

# Frontend
npm run dev
```
